# NBA-Analytics

Publish my NBA analytical projects


## **1. NBA Top 20 All-Time Points Leaders - Regular Season VS Playoffs** - [Link](http://rpubs.com/Juanma7/431499)

NBA Stats Analysis of the Top 20 All-time Points Leaders.  Stats for Regular Season and Playoffs where we can see how much each player contributes to the different games in which they played.

This analysis contains data until October 21st 2018.

I've used the d3heatmap which easily represents all the players stats. We can hover over on it to see the name of the player, the stats and value.

## **2. Wins comparison over the last four seasons**  - [Link](http://rpubs.com/Juanma7/431945)

In this analysis, we can compare the number of games won by each team by conference over the last four seasons.

We can see what teams

- performed more consistently over the last four seasons with similar number of games won.
- showed and upward/downward trend.

I've used the plotly package for the visualization.
